DESCRIPTION

This module creates the grid information needed for the SWIM netCDF climate input module if called with the ncinfopath and grid arguments. The lon-lat grid size is set via the res argument (default is 0.5 degree). An arbitrary grid can be defined by parsing it to grid witht the -d flag.

All other arguments are for creating the stationfile and virutalstationfile for the R climate data interpolation script (von T. Vetter).
